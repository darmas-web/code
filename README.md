# Sistema de gestão da Clinica Darmas
Repositório de código do sistema desenvolvido para clinica darmas.

## Tecnologias:

* Python 3.6.1
* Flask 1.0.2
* PostgreSQL
* Bulma Framework
* Vuejs


Ambiente de testes (local):
* Vagrant
* Ubuntu 14.04
* Dokku

Hospedado na digitalocean, usando:

* Droplet(VM) linux
* Dokku
